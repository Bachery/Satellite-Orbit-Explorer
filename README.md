# Satellite Orbit Explorer

一个使用 Svelte + Three.js 构建的教学型轨道六根数可视化工具。它用于直观看到 `a / e / i / Ω / ω / ν` 如何共同决定轨道形状、空间方向和卫星当前位置。

## 运行方式

```bash
npm install
npm run dev
```

然后打开终端输出中的本地地址，通常是 `http://127.0.0.1:5173/`。

生产构建：

```bash
npm run build
npm run preview
```

## 已实现功能

- 3D 地球、地轴、地心坐标轴、赤道面、轨道面。
- 语言选择：中文 / English，主要页面文案可即时切换。
- 标准六个开普勒轨道要素控制：半长轴 `a`、偏心率 `e`、倾角 `i`、升交点赤经 `Ω`、近地点幅角 `ω`、真近点角 `ν`。
- 轨道曲线、近地点/远地点标记、卫星位置、速度方向箭头实时刷新。
- 派生量面板：近地点/远地点距离与高度、轨道周期、当前地心距离、vis-viva 速度、半通径。
- 参数联动高亮：调整不同参数时突出轨道、轨道面、升交点线、近地点方向或卫星半径线。
- 图层开关、动画播放/暂停、动画速度、动画模式、相机快捷视角。
- 6 个 preset：低轨圆轨道、高偏心椭圆、极轨、太阳同步近似、GEO、Molniya 风格。
- 对特殊情形给出提示：圆轨道中 `ω` 退化、赤道轨道中 `Ω` 退化、近地点进入地球内部。

## 数学实现

核心公式位于 `src/lib/orbitMath.js`：

- 地球半径：`R_earth = 6378 km`
- 地球引力参数：`mu = 398600.4418 km^3/s^2`
- 半通径：`p = a * (1 - e^2)`
- 半径：`r = p / (1 + e * cos(ν))`
- 近地点/远地点：`r_p = a * (1 - e)`，`r_a = a * (1 + e)`
- 周期：`T = 2pi * sqrt(a^3 / mu)`
- 速度：`v = sqrt(mu * (2/r - 1/a))`

轨道点先在 perifocal 轨道平面内生成，再按 `Rz(Ω) * Rx(i) * Rz(ω)` 旋转到地心惯性参考系。应用采用 `Z` 轴作为地轴，`XY` 平面作为赤道面。

## 工程判断与教学近似

- 动画提供两种模式：固定角速度模式让 `ν` 匀速推进；两体速度模式按平均运动推进平近点角 `M`，解开普勒方程后得到 `ν`，因此近地点更快、远地点更慢。
- 两体速度模式使用 `600x` 的时间压缩，让真实周期差异可以在教学场景中看见；它保留两体相对速度关系，但不是实时墙钟速度。
- 太阳同步 preset 只展示接近太阳同步轨道的几何倾角，没有加入 J2 摄动，因此不是动力学精确模型。
- 当 `e` 接近 0 时仍保留 `ω` 控件，方便比较退化前后的几何变化。
- 当 `i` 接近 0 deg 或 180 deg 时仍保留 `Ω` 控件，方便观察升交点定义失去唯一性。
- 即使近地点高度低于 0，应用也允许继续显示几何结果，但会明确提示该轨道不物理可行。
